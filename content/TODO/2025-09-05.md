## 1️⃣ How do you ensure the performance of a web application?

（你如何保证 Web 应用的性能？）

> To ensure good performance, I usually focus on three areas:
> 
> 1. **Code splitting and lazy loading** to reduce the initial bundle size.
>     
> 2. **Caching and CDN** to speed up asset delivery.
>     
> 3. **Optimizing rendering** by minimizing reflows and using virtualized lists for large datasets.
>     
> 
> In one of my previous projects, I applied code-splitting with dynamic imports, which reduced the initial load time by around 30%.

---

## 2️⃣ Can you explain the difference between Vue and React, and when you would use each?

（Vue 和 React 的区别，以及使用场景？）

> Vue provides a more opinionated structure with templates and built-in solutions like directives, which makes it easy to start quickly.  
> React, on the other hand, is more flexible and relies heavily on JavaScript, which gives developers more control but requires more setup.
> 
> If I need a **fast MVP or small-to-medium project**, I’d usually choose Vue for its simplicity.  
> For a **large-scale project with a bigger ecosystem**, React might be a better choice due to the wider community and third-party support.

---

## 3️⃣ How do you handle cross-browser compatibility issues?

（你如何处理浏览器兼容性问题？）

> I usually start by using **modern standards and polyfills** when needed.  
> I also rely on tools like **Babel** and **PostCSS** to ensure code runs on different browsers.  
> In addition, I use testing tools like **BrowserStack** to verify compatibility.
> 
> For example, in one project we had issues with Flexbox in older versions of IE. I solved it by using fallback styles and ensuring graceful degradation.

---

## 4️⃣ Can you describe a challenging project you worked on, and how you solved the problems?

（描述一个有挑战性的项目，以及你是怎么解决的。）

> In one of my recent projects, we had performance issues with rendering a large dataset on the page. The app became slow when handling more than 10,000 records.
> 
> I researched and implemented **virtualized rendering** with libraries like Vue Virtual Scroll List / React Virtualized.  
> This reduced memory usage significantly and improved rendering performance.
> 
> As a result, the page load time dropped from 6 seconds to less than 2 seconds, and the user experience improved a lot.

---

## 5️⃣ How do you usually collaborate with designers and backend developers?

（你通常如何和设计师、后端合作？）

> I usually communicate frequently and keep alignment through tools like Figma for design handoff, and Swagger/Postman for API contracts.  
> With designers, I give feedback on feasibility and performance considerations.  
> With backend developers, I align on API structure and error handling early to avoid rework.
> 
> I believe good communication upfront saves a lot of time later in the development cycle.
