### 概览
```dataviewjs
// 获取所有符合条件的页面并转换为数组
const pages = Array.from(dv.pages('#装修').where(p => p.purchase_price));

// 显示表格
if (pages.length > 0) {
    dv.table(["项目", "价格"], 
        pages.sort((a, b) => a.file.name.localeCompare(b.file.name))
            .map(p => [
                dv.fileLink(p.file.name, false, p.product_name || p.file.name),
                p.purchase_price + "元"
            ])
    );

    // 计算并显示总计
    const total = pages.reduce((sum, p) => sum + (p.purchase_price || 0), 0);
    dv.paragraph("**总花费: " + total.toLocaleString() + "元**");
} else {
    dv.paragraph("暂无购买记录");
}
```
